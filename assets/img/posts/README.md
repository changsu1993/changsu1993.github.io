# 포스트 이미지 가이드

이 디렉토리에 블로그 포스트의 썸네일 이미지를 저장하세요.

## 이미지 준비 방법

### react-hooks-optimization.jpg

**추천 이미지 소스:**
1. [Unsplash](https://unsplash.com/) - 무료 고품질 이미지
   - 검색어: "code", "programming", "developer", "react"
2. [Pexels](https://www.pexels.com/) - 무료 스톡 이미지
3. [Canva](https://www.canva.com/) - 직접 디자인

**이미지 사양:**
- 권장 크기: 1200 x 630px (OG 이미지 표준)
- 포맷: JPG 또는 PNG
- 파일 크기: 500KB 이하 (최적화 권장)

**이미지 최적화 도구:**
- [TinyPNG](https://tinypng.com/) - 이미지 압축
- [Squoosh](https://squoosh.app/) - Google의 이미지 최적화 도구

## 이미지 추가 후

이미지를 추가한 후에는 다음 명령어로 커밋하세요:

```bash
git add assets/img/posts/react-hooks-optimization.jpg
git commit -m "feat(post): React Hooks 최적화 포스트 이미지 추가"
git push origin main
```
